# Assets Folder

This folder contains all static assets for the Qlaat Al Madeena Transport website.

## Folder Structure

- **images/** - All image files (logos, photos, banners, etc.)
- **icons/** - Icon files (SVG, PNG icons)
- **fonts/** - Custom font files (if any)

## Usage

Assets in this folder can be accessed in your code using:

```jsx
// Example: Using an image
<Image src="/assets/images/logo.png" alt="Logo" />

// Example: Using an icon
<img src="/assets/icons/truck.svg" alt="Truck Icon" />
```

## Notes

- All files in the `public` folder are served statically
- Use optimized images (WebP, optimized PNG/JPG) for better performance
- Keep file sizes reasonable for faster page loads
